# Netflix EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Netflix Movies and TV Shows data.

## Objective
The goal is to analyze Netflix content and discover trends in:
- Movies vs TV Shows
- Content Ratings
- Countries
- Genres
- Release Years

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Netflix Movies and TV Shows Dataset

## Key Insights
1. Movies dominate Netflix content.
2. Most content was added after 2015.
3. United States produces the most content.
4. TV-MA is the most common rating.
5. Drama is one of the most popular genres.
